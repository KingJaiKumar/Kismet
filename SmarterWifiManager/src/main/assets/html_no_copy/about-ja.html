迷ったときに、スマートに

Smarter Wi-Fi Manager はスマートになることを目指します - 一般的に、Wi-Fi 状態の管理はバックグラウンドで行われ、 見えないところで行われるべきです。

自動学習の使用

Smarter Wi-Fi Manager は、どの基地局の近くで Wi-Fi を使用しているか、自動的に学習します。

初めて新しい場所に移動したときは、手動で Wi-Fi をオンにするだけです。 正常にネットワークに接続するとすぐに、Smarter Wi-Fi Manager は近くの基地局を学習します。

Wi-Fi ネットワークの範囲を離れると、Smarter Wi-Fi Manager は自動的に Wi-Fi 無線をオフにします。

以前学習した場所に戻ると、Wi-Fi は自動的に有効になります。

学習したネットワークのメンテナンス

時間が経つにつれて、Wi-Fi ネットワークに接続された場所の数が、コントロールできないくらいに 増えることがあります。 "位置メンテナンス" プリファレンスを有効にすると、 パフォーマンスに影響を与えることなく、学習済の位置のリストを小さく保持します。

基地局のメンテナンスは定期的に発生しますが、 あなたが Wi-Fi に接続したときだけ です。 言い換えると、Smarter Wi-Fi Manager が Wi-Fi ネットワークから位置を削除するのは、 予期しない動作を最小限に抑えることができる、すぐに新しい場所を学習することができるとき です。

ネットワークの無視

一部のネットワーク - ケーブル会社によって経営されているような、または 'linksys' のようなデフォルトのアクセスポイント名 - は、いつも表示されます。これらのネットワークは、 自動学習システムを混乱させ、望まれる以上に頻繁に Wi-Fi をオンにしたままにすることがあります。

無視リストにネットワークを追加すると、そのネットワーク名に接続している間は、自動学習をオフにします。 ネットワークを切断するか、別のネットワークに変更することにより、そのネットワークを離れると、 学習は自動的に再度有効になります。

Bluetooth ブラックリスト

Smarter Wi-Fi Manager は、指定された Bluetooth デバイスに接続したときに、Wi-Fi をオフにすることができます。

例えば、あなたがカーステレオに接続するたびに Wi-Fi ネットワークをオフにしたい場合があります。 ブラックリストに載せたデバイスに接続している間、Wi-Fi はオフのままになり、学習が無効になります。
